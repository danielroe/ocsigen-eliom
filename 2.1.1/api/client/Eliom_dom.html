Module Eliom_dom 

module Eliom_dom : sig..end

Dom manipulation.

In this module, all the functions apply only to HTML5 element with Error a_manual: exception Failure("no manual given") .

Dom manipulation.

In this module, all the functions apply only to HTML5 element with Error a_manual: exception Failure("no manual given") . 

val appendChild : 
  ?before:'a Eliom_pervasives.HTML5.elt ->
  'b Eliom_pervasives.HTML5.elt -> 'c Eliom_pervasives.HTML5.elt -> unit

The function appendChild e1 e2 inserts the element e2 as last child of e1. If the optional parameter ~before:e3 is present and if e3 is a child of e1, then e2 is inserted before e3 in the list of e1 children.

val appendChilds : 
  ?before:'a Eliom_pervasives.HTML5.elt ->
  'b Eliom_pervasives.HTML5.elt ->
  'c Eliom_pervasives.HTML5.elt list -> unit

The function appendChilds e1 elts inserts elts as last children of e1. If the optional parameter ~before:e3 is present and if e3 is a child of e1, then elts are inserted before e3 in the list of e1 children.

val removeChild : 
  'a Eliom_pervasives.HTML5.elt -> 'b Eliom_pervasives.HTML5.elt -> unit

The function removeChild e1 e2 removes for e2 from the list of e1 children.

val replaceChild : 
  'a Eliom_pervasives.HTML5.elt ->
  'b Eliom_pervasives.HTML5.elt -> 'c Eliom_pervasives.HTML5.elt -> unit

The function replace e1 e2 e3 replaces for e2 by e3 in the list of e1 children.

val removeAllChild : 'a Eliom_pervasives.HTML5.elt -> unit

The function removeAllChild e1 removes e1 children.

val replaceAllChild : 
  'a Eliom_pervasives.HTML5.elt ->
  'b Eliom_pervasives.HTML5.elt list -> unit

The function replaceAllChild e1 elts replaces all the children of e1 by elt.

val addEventListener : 
  ?capture:bool ->
  'a Eliom_pervasives.HTML5.elt ->
  (#Dom_html.event as 'b) Js.t Dom_html.Event.typ ->
  ('a Eliom_pervasives.HTML5.elt -> 'b Js.t -> bool) ->
  Dom_html.event_listener_id

The function addEventListener elt evt handler attach the handler for the event evt on the element elt. See the Js_of_ocaml manual, for a list of available events.

module Named : sig..end

Dom manipulation by element identifier.

val scrollIntoView : ?bottom:bool -> 'a Eliom_pervasives.HTML5.elt -> unit

The function scrollIntoView elt scroll the page to a position where elt is displayed at the top of the window. If the optional parameter ~bottom:true is present, the page is scrolled to a position where elt is displayed at the bottom of the window.
